# Music Player

A **dark-themed, lightweight music player** built with [Electron](https://www.electronjs.org/).  
Play your favorite tracks directly on your computer with a sleek interface.

---

## Features

- Dark-themed interface for comfortable night-time use  
- Play audio files locally using **Howler.js**  
- Read metadata from music files using **music-metadata**  
- Customizable icons and easy-to-use controls  
- Cross-platform support via Electron
